// Cognitive Stack
The 4-Tier Memory Hierarchy
Working Memory (RAM)

Active context window. Fast reasoning, low persistence.

LATENCY: <10ms VOLATILE
Episodic Memory (The Log)

Append-only record of past interactions and raw experiences.

LATENCY: ~200ms DURABLE
Semantic Memory (The KB)

Distilled facts, entity relationships, and world models.

LATENCY: ~500ms EVOLVING
Procedural Memory (Skills)

Successful workflow patterns and tool-use heuristics.

LATENCY: ~1s LEARNED
