# AI Memory Content Strategy (March 2026)

## The Landscape
The AI memory space has matured from simple vector search (RAG) into dedicated "Agentic Memory" systems. The market is divided into three tiers:
1. **Specialized Memory Layers:** Letta (formerly MemGPT), Zep AI, Cognee (Knowledge Graphs), Hindsight, Open Viking.
2. **Stateful Frameworks:** LangGraph, OpenAI Agents SDK, Google ADK, CrewAI, AutoGen. 
3. **Infrastructure (Vector/Graph):** Pinecone, Qdrant, Weaviate, LanceDB (Edge), pgvector.
