Classes of Production Agent Failure
Class 01 / Ambiguous
Tool double-execution on retry
Handled byIdempotent tools or a dedup layer keyed on operation_id.
Class 02 / State
Silent state corruption
Handled byExplicit state transitions + checkpoint before ambiguous retry.
Class 03 / Context
Context window exhaustion
Handled byHard limit on tool-call history, fail explicitly near the cap.
Class 04 / Schema
Tool schema drift
Handled byVersion schemas, validate inputs against expected version.
Class 05 / Parallel
Partial failure in parallel calls
Handled byTrack each call independently, no all-or-nothing assumption.
Class 06 / Observability
Observability gaps
Handled byLog every tool call: input, output, latency, error.
