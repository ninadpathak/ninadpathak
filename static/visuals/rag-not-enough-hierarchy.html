Agent Memory Hierarchy
Layer 01RAG MISSES
Working context
What the agent reasons about right now. Lives in the context window. Lost on every reset.
Layer 02RAG MISSES
Episodic recall
Attributed observations from past turns. Not documents. Time-sensitive. Needs a structured log.
Layer 03RAG COVERS
Semantic / document layer
World knowledge + persistent facts. Vector search over embeddings. The layer RAG was built for.
Layer 04RAG MISSES
Cross-session persistence
Facts that survive past the session. RAG pipelines are session-scoped and evaporate at the end.
RAG covers
1 of 4 layers
RAG misses
working, episodic, cross-session
A vector store is one piece. The other three layers need a structured episodic store + working-context manager.
